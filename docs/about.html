About Subspecies Mapper

This website displays maps of subspecies eBird reports for 1311 species with reportable subspecies on eBird. Sensitive species and species without any subspecies reported to eBird are not included.

Terminology

For a given species, maps display all reported "infraspecies" of the three following categories, following the eBird taxonomy:

  • Subspecies: Taxonomic units follow subspecific boundaries as defined by the Clements checklist. May include groups of 2+ subspecies
  • Forms: Catch-all for forms without a scientific name, including identifiable groups, unrecognized groups that may represent distinct taxa, etc.
  • Intergrades: Birds descended from multiple subspecies, forms, or both

Data downloads

Want to create your own maps? Tables of subspecies data used to create these maps are available here (see usage requirements below).

Maps for each species are separated by:

  • H3 resolution: either resolution 2, 3, or 4. See H3 documentation for more information.
  • Report status: either "unflagged", reports that were not originally flagged by eBird filters, or "flagged and accepted" reports that were confirmed by eBird reviewers.

To use these data, you must agree to the eBird Data Terms of Use. Please cite this website as well as the eBird dataset if you use these data.
